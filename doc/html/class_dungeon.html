void Dungeon::handleEvent ( SDL_Event *  event)
overridevirtual

If the cards are selectable (the dungeon is "activated"), checks if the player clicked on a card and passes the event to the found card. If the dungeon is not activated because a BigCard is displayed, passes the event to the displayed BigCard.

Parameters
eventthe mouse event of the current player.
